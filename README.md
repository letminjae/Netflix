## 기술스택
- Programming Language : `Typescript` 
- FrontEnd Framework : `React`
- Style library : `Styled-components`
- Animation library : `Framer-motion` 
- State Management : `Recoil`
- Data Management : `React-query`
- Form library : `React-Hook-Form`
- Deploy : `AWS S3`
